# Elysia Gemini Wrapper
### by : Roy


bikin file `.env` di root isinya ini : 

```
BUN_ENV=production 
GEMINI_API_KEY='apikey mu'
PORT=3042
```

### install package

```bash
bun install 
```

### jalankan aplikasi

```bash
bun run dev
```

dibuat dengan [Elysia](https://elysiajs.com/) dan [Gemini API](https://ai.google.dev) by [Roy](https://github.com/idlanyor)
